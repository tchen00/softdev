What is Ionic Framework?

  • Framework for building mobile/desktop apps using HTML, CSS, and JS
  • Focused on frontend user experience & UI interaction of an app
  • Can be used and integrated with Angular and React or used alone with a simple script

Why Ionic Framework?

  • Open-source & free
  • Cross-platform (native iOS, Android, desktop, web)
  • Clean, simple, and functional design
  • Support both linear & non-linear app navigation
    • Linear → user navigates forward to a page and can hit back to navigate back
    • Non-Linear → user never loses their place as they navigate and switch between tabs
  • Can be used with Capacitor and Cordova to give Ionic apps access to SDKs

Ionic Framework Installation

  1. install node.js ⇒ NPM
  2. sudo npm install -g @ionic/cli
  3. sudo npm install -g cordova
  4. start project:
    • ionic start [name] [template] --type ionic1
    • ionic serve
    • ionic serve -l
    • ionic serve --address <droplet_ip>

How To Test The Ionic App:

  1. Desktop Browser
    • ionic serve [options]
  2. Simulator
    • ionic cordova build ios/android
    • ionic cordova emulate ios/android
    • On iOS: Xcode
  3. Mobile Browser
    • Load IP address of your computer in a browser on your phone
    • Furthest from actual experience
  4. Native App
    • Apple Developer - $99/year
    • Plug in android phone
      • ionic cordova run android

Examples of Ionic Framework Components

Fabs

Floating Action Button (FAB) are circle shaped icons floating above the UI. It is a tool for designers to call out key parts of their applications.

Tabs

This is helpful if you are looking to build a tab-based navigation app.

Lists

In Ionic, lists support several interactions such as swiping items, dragging to reorder items, and deleting items.
